# HackMol 5

A clean and modern website for the fifth edition of HackMol, designed to provide all important event details in one place.

## About

HackMol 5 is a hackathon that brings together developers, designers, and innovators to build impactful solutions.
This website acts as the central hub for announcements, registrations, and event updates.
